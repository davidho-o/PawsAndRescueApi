# 🐾 Paws & Rescue - Animal Shelter API

Welcome to the backend repository for the **Paws & Rescue** management system.
This application helps a local animal shelter manage its volunteers, shifts, and daily tasks efficiently.

## 👥 Users

The system supports two types of users:

- **Volunteer**: Can sign in, read safety regulations, choose shifts, and check task instructions (e.g., feeding dogs, cleaning cages).
- **Manager**: Can approve shifts, manage volunteer profiles, and add new dog profiles.

## 🗄️ Database Models (Planned)

- `Users` (Volunteers & Managers)
- `Dogs` (Profiles, Temperament, Notes)
- `Shifts` (Scheduling and Cover-ups)
- `Tasks` (Daily shelter activities)

## 💻 Tech Stack

- **Backend:** .NET 10 (C#) Web API
- **Database:** Entity Framework Core (SQLite for development)
